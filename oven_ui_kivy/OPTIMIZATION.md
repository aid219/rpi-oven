## 📊 Текущая проблема

**Симптом:** Лаги при перемещении по интерфейсу

**Причины:**
1. Анимации частиц на фоне (5 объектов + анимации)
2. Переходы между экранами с анимацией
3. Частые обновления canvas (перерисовка)
4. SDL2 + OpenGL ES накладные расходы

## 🔧 Дополнительные оптимизации

### 1. Отключить vsync (уменьшает input lag)

В `main_optimized.py`:
```python
Config.set('graphics', 'vsync', '0')  # Раскомментировать
```

### 2. Уменьшить сложность кнопок

В `screens/main_screen.py`:
```python
# Упростить AnimatedButton
# Меньше свойств для анимации
```

### 3. Использовать Texture для кнопок

Вместо перерисовки canvas использовать готовые текстуры:
```python
from kivy.core.image import Image
texture = Image('button.png').texture
```

### 4. Отключить сглаживание

В `main_optimized.py`:
```python
Config.set('graphics', 'multisamples', '0')
```

### 5. Уменьшить количество виджетов

Каждый виджет — это draw call. Меньше виджетов = выше FPS.

### 6. Использовать Cache

```python
from kivy.cache import Cache
Cache.register('my_cache')
```
