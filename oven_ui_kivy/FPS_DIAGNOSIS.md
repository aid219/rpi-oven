# 🎯 РЕАЛЬНАЯ ПРОБЛЕМА: 16 FPS

## 🔍 Диагностика

```
FPS: 16.6 (avg:1600, min:16.6, max:3461)
```

**Реальный FPS: 16.6** — это **физическое ограничение SPI дисплея**!

---

## ❌ Почему 70-80 FPS в benchmark?

`fps_benchmark.py` показывает **70-80 FPS** потому что:
- Считает **внутренний FPS** Kivy (сколько кадров отрендерено)
- **НЕ** учитывает обновление дисплея

**Реальный FPS** = сколько раз **дисплей обновился** = **16.6**

---

## 🔧 Почему движение "рваное"?

**16 FPS** — это **мало** для плавного движения:
- ✅ 60 FPS — идеально плавно
- ⚠️ 30 FPS — нормально
- ❌ 16 FPS — видно рывки
- ❌ <10 FPS — слайдшоу

---

## 💡 Решения

### 1. Принять ограничение (просто)

SPI дисплей **физически** не может больше 16-17 FPS.

**Что делать:**
- Использовать более простые анимации
- Увеличить длительность переходов (0.5-1 сек)
- Меньше движения = меньше заметно

### 2. Прямая запись во framebuffer (сложно)

Kivy использует OpenGL → DRM → SPI (медленно)

**Альтернатива:**
```python
# Прямая запись в /dev/fb0
# Как в fb_animation.py
fb = open('/dev/fb0', 'wb')
fb.write(data)
```

**Преимущества:**
- Меньше накладных расходов
- Больше контроль

**Недостатки:**
- Нет аппаратного ускорения
- Нужно писать всё с нуля

### 3. Использовать Pygame + FB (компромисс)

```bash
# Вернуться к fb_animation.py
# Pygame + прямая запись в framebuffer
python3 ~/fb_animation.py
```

**FPS:** ~14-15 (то же ограничение)

### 4. Другой дисплей (радикально)

Если нужен **плавный UI (60 FPS)**:
- HDMI дисплей
- DPI (parallel) дисплей
- Не SPI

## ✅ Рекомендация

**Для SPI дисплея 320x480:**

1. **Используйте Kivy** с `main_optimized.py`
2. **Увеличьте длительность анимаций** до 0.5-0.7 сек
3. **Избегайте быстрого движения** (скролл, drag)
4. **Принимайте 16 FPS** — это физический предел

**Код оптимизации:**
```python
# Более плавные переходы на 16 FPS
```

## 📝 Вывод

**Проблема НЕ в коде** — проблема в **железе** (SPI дисплей).

**16 FPS** — это **максимум** для SPI на Raspberry Pi Zero 2 W.

**Решение:** Принять ограничение и оптимизировать UI под 16 FPS.
