# 🔧 Тест переходов между экранами

## Проблема
При переходе между экранами видно отрисовку **кусками** (артефакты).

## Причина
SPI дисплей + Kivy анимация = **partial updates** во время перехода.

---

**Ожидание:** Плавная анимация 0.5 сек, возможны артефакты.

## 🎯 Что делать

### Если main_no_transition.py работает без артефактов:

**Проблема** в том что SPI дисплей не успевает за анимацией.

**Решения:**

1. **Использовать NoTransition** (быстро, но не красиво)
   ```python
   self.sm.transition = NoTransition()
   ```

2. **Увеличить длительность перехода** (медленнее = меньше артефактов)
   ```python
   self.sm.transition = FadeTransition(duration=0.5)
   ```

3. **Снизить сложность сцены** во время перехода
   - Убрать анимации
   - Упростить canvas
   - Меньше виджетов

4. **Прямая запись во framebuffer** (как в fb_animation.py)
   - Kivy не оптимален для SPI
   - Pygame + прямой доступ к /dev/fb0
